Voxel Road는 Crossy Road 모작 프로토타입이다. 캐릭터 수와 게임 아트에 공을 들이지 말 것. 핵심 목표는 **게임이 정상적으로 실행·구동되고 버그 없이 동작하는 것**.

**Why:** 1인 개발·프로토타입 범위. 상용 Crossy Road의 수백 종 캐릭터·가챠·코인·광고 제거 수익 모델 같은 상용화 레이어는 스코프 밖. CLAUDE.md에도 "아트 확장성은 고려 대상 아님"으로 명시됨(사용자가 2026-04-23 재확인).

**How to apply:**
- 기능 제안 시 "정상 구동에 필요한가?" 먼저 묻고, 캐릭터 수집/아트 확장/비주얼 폴리시는 제안하지 말 것
- 로드맵 범위 내에서만 작업 제안. 현재 남은 Step 8~10(UI·점수/난이도/오디오). 코인·가챠·캐릭터 다수화는 제외
- 구조가 예뻐도 버그 있으면 실패 — 버그 픽스·안정화 우선순위가 신기능보다 항상 높음
- Kenney 에셋 이상의 아트 파이프라인 투자 제안 금지(쉐이더·라이팅·포스트프로세스 확장 등)
- 독수리(Step 7 원안)는 폐기됨 — CC0 에셋 부재 + 시각 표현 없는 추격자는 몰입 약화. 자동 전진 카메라+추격 벽으로 대체 완료(2026-04-27)
- **2026-05-04 추가 명확화**: 출시 목적 X — 포트폴리오 시연용. Google Play 등록·상업 판매 안 함. 출시 절차(키스토어·메타데이터·Android 실기 검증)는 모두 스코프 밖. 시연 경로 = WebGL 빌드 → `https://bomb1961.github.io/Voxel-Road/` 호스팅(방법 3, gh-pages 브랜치). 사이트 본체(`BomB1961.github.io`)에는 게임 페이지 링크/iframe만 추가(사용자 직접).
