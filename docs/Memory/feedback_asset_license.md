외부 에셋 추가 시 **CC0 (Creative Commons Zero / Public Domain) 만 사용**. CC-BY 이상(SA/NC/ND 포함) 전부 제외.

**Why:** 1인 개발·상업 배포 예정. CC-BY 이상은 크레딧 관리·법적 리스크 관리 부담 존재. Kenney 등 CC0 에셋으로 충분히 커버 가능한 범위임.

**How to apply:**
- Kenney.nl / Quaternius / poly.pizza CC0 필터 / OpenGameArt CC0 태그만 사용
- 다운로드 전 License.txt 확인 → CC0 아니면 제외
- 새 에셋 추가 시 `Assets/ThirdParty/README.md` 라이선스 표에 출처·URL 기록
- 꼭 CC-BY를 써야 한다면 사용자 승인 먼저 받기 (예외 케이스)
- `Assets/ThirdParty/README.md` 라이선스 규칙 섹션이 진실 — 의심 시 파일 확인

**예외 승인 기록:**
- **Sonniss GameAudioGDC** (2026-05-04 승인): 자체 라이선스로 게임 상업 사용·royalty-free·attribution 의무 없음. 단 사운드 라이브러리 재배포 금지. Step 10 오디오 음원 소스로 사용. CC0는 아니지만 게임 임베드 용도로는 동등하게 안전 — Hades 등 다수 게임이 사용.
